# Hotel_Nebula

Projeto de banco de dados não relacional desenvolvido para modelar e gerenciar as operações do Hotel Nebula (fictício) utilizando MongoDB.

## Objetivo

Criar uma estrutura de dados capaz de armazenar e organizar informações sobre hóspedes, quartos, reservas, hospedagens, pagamentos, funcionários, serviços e avaliações, permitindo consultas e análises para apoio à gestão do hotel.

## Sobre o Projeto

O desenvolvimento foi dividido em três etapas:

* Modelagem do banco de dados.
* Implementação das coleções e documentos no MongoDB.
* Criação de consultas para extração de informações gerenciais.

Os arquivos deste repositório contêm a estrutura das coleções, documentos de exemplo e consultas desenvolvidas para atender aos requisitos da atividade.

## Tecnologias Utilizadas

* MongoDB Atlas
* MongoDB Compass
* JSON
* GitHub
